1. Beyond Unit Tests

Unit tests verify individual components. But AI agents are non-deterministic, agentic systems that use tools, maintain state, and handle long-running conversations. You need agent-based evaluation.

2. The 3-Tier Eval Framework

  • Unit Evals: Mocking tools for basic component verification.
  • Scenario Evals: Testing agents in controlled multi-turn scenarios (e.g., "Customer says X, Agent should do Y or Z").
  • Production Evals (Shadow Mode): Running agents against production data without taking actions (using the OPA — Observability Pattern).
